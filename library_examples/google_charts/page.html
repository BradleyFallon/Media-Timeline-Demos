Video Timeline: Google Charts Implementation

Return to the main page

This example shows how to use the Google Charts Timeline to display a video timeline.

Hover on a bar to see the event details.

Notes

Google Charts is a good library for creating basic timelines. It is easy to use and is not bloated with features that you don't need. It does not have support for making markers as points, but you can use zero-length bars to achieve the same effect.
